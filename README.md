# SolidWorks Automation using Python (Create Circle)

This project demonstrates how to automate SolidWorks
to create a circle using Python and COM API.

## Description
- Connects Python with SolidWorks using pywin32
- Automatically creates a sketch
- Draws a circle without manual interaction

## Technologies Used
- Python
- SolidWorks COM API
- pywin32

## Requirements
- SolidWorks 2021 or later
- Python 3.x
- pywin32 library

## How to Run
1. Open SolidWorks
2. Run the Python script
3. A circle will be created automatically in a sketch

## Use Case
This project is useful for:
- CAD automation
- Design parameterization
- Reducing repetitive manual work
